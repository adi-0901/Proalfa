2. Physical File Management

Every employee must have a physical file maintained in a dedicated folder/binder. Keep files locked in a cabinet. Label each file with the employee name, ID, and date of joining.

What Goes in Each Employee File

  • Aadhar Card — Self-attested copy. Mandatory.
  • PAN Card — Self-attested copy. Required for TDS. If PAN is not available, TDS is deducted at 20%.
  • Passport Photo — Minimum 2 copies. One for ID card, one for file.
  • Bank Account Details — Cancelled cheque or passbook front page copy. Required for salary transfer.
  • Educational / Skill Certificates — Original verification not required, attested copies acceptable.
  • Previous Employment Documents — Relieving letter or experience letter if applicable.
  • Appointment Letter (signed copy) — One copy stays in the file, one goes to the employee.
  • Offer Letter (if issued separately)
  • PF Declaration Form (Form 2) — For nominees. Required when enrolling in PF.
  • ESIC Declaration — If employee is ESIC-eligible (gross ≤ ₹21,000).
  • Emergency Contact Form — Name, relationship, phone number.
  • Increment Letters — Filed chronologically.
  • Warning / Show Cause Letters — If any disciplinary actions taken.
  • Resignation / Termination Letter — Filed at exit.
File Tip: Use a plastic document separator inside each file to divide sections: (1) KYC Docs, (2) Employment Letters, (3) Compliance Forms, (4) Disciplinary/Exit. This makes audits easy.

File Register

Maintain a simple File Register — a spreadsheet or ruled notebook — listing every employee, their file number, and the checklist of documents received vs pending. Cross-check this with the ERP's onboarding checklist monthly.

4. Statutory Deductions Reference

These are the deductions the ERP calculates. Understand what each is so you can explain salary slips to employees.

DeductionWho it Applies ToRateNotes
PF (Employee) PF-enrolled employees 12% of Basic Salary Capped at ₹15,000 basic unless "actual" wage cap is set. Goes to employee's EPF account.
PF (Employer) PF-enrolled employees 12% of Basic Salary Company's contribution. Not deducted from salary — it's an additional cost to company.
ESIC (Employee) Gross ≤ ₹21,000/month 0.75% of Gross Provides medical insurance. Employer contributes 3.25% additionally.
Professional Tax (PT) PT-applicable employees ₹200/month Maharashtra state tax. Applicable to most employees above a threshold.
TDS Tax-liable employees (higher salary) As computed by CA Editable per employee in salary sheet. Default 0. Update monthly if applicable.
Advance Recovery Employees with outstanding advances As agreed Log in Advances tab. ERP auto-deducts in that month's salary.

5. Employee Relations & Discipline

Attendance Issues

  • Habitual late arrivals (more than 3 times/month): Issue a verbal warning first, then written warning if repeated.
  • Unapproved absences: Mark as unpaid. Notify the employee in writing if it becomes a pattern.
  • No-show without information for 3+ continuous days: Send a Show Cause Notice (SCN) by registered post and WhatsApp.

Leave Abuse

  • Employees get 1 paid leave per month. Any additional absence is unpaid.
  • If sick leave is suspected to be false, you may request a medical certificate for absences of 2+ days.
  • Consistent Monday/Friday absences are a red flag — track in ERP monthly view.

Disciplinary Process

  1. Verbal Warning — Documented internally by HR, noted in employee file.
  2. Written Warning — Formal letter, employee signs acknowledgement. Copy in file.
  3. Show Cause Notice (SCN) — Employee must respond in writing within 48–72 hrs.
  4. Domestic Enquiry — If the matter is serious (theft, misconduct). Requires a proper enquiry officer.
  5. Termination — Only after due process. Give notice or pay in lieu as per appointment letter.
Rule: Never terminate an employee verbally or without documentation. All disciplinary actions must go through at least a Show Cause Notice and the employee's written response. This protects the company from labour disputes.

Exit Formalities

  • Collect resignation letter in writing (WhatsApp screenshot acceptable for workers).
  • Serve notice period as per appointment letter (7 days probation / 30 days post-confirmation) or recover notice pay from full and final settlement.
  • Complete No-Objection Checklist: PPE returned, tools returned, ID card returned, pending advances cleared.
  • Prepare Full & Final Settlement — unpaid salary + OT + leave encashment (if applicable) minus any recoveries.
  • Issue Relieving Letter and Experience Letter within 15 days of last working day.
  • Update ERP: change employee status to "resigned" or "terminated" in HR Dashboard.
  • Remove employee from ZKTeco attendance machine.

Step 3 — Setting Up Attendance

After adding the employee in HR Dashboard, sync them to the Attendance system.

Attendance & Salary › Employees tab › ↻ Sync from HR
1
Open Attendance & Salary. Go to Employees tab and click ↻ Sync from HR. This pulls all active employees from HR Dashboard into the attendance system.
2
Verify the new employee appears in the list with correct designation, type, and OT eligibility.

ZKTeco Machine Setup

3
On the ZKTeco K40 Pro machine: go to User Management → Add User. Set the User ID to exactly match the Employee ID in the ERP (e.g., if ERP ID is 7, set machine User ID to 7).

Enrol the employee's fingerprint (enrol 2 fingers — index and middle — for reliability).

Critical: The Employee ID in ERP and the User ID on the ZKTeco machine must match exactly. If they don't match, that employee's machine punches will be ignored by the sync script.

Step 4 — Daily Attendance

What Gets Auto-tracked

  • Machine syncs every minute via cron. Records appear in Today view within 1–2 minutes of punching.
  • The system auto-classifies: Present Half Day Late based on punch times.
  • Check-in after 10:00 AM → flagged Late.
  • Check-in after 12:00 PM or check-out before 3:00 PM → Half Day.
  • Check-out after 6:00 PM → OT hours calculated automatically.
  • Punches before 5:00 AM are attributed to the previous day's shift (for night-shift OT).

Your Daily Routine

1
Each morning, open Today tab in Attendance & Salary. Check for absent employees and note names.
2
Check ✅ Approvals tab for pending site attendance requests. For each request: click the 📍 location link to verify the employee is genuinely on site. Approve or reject accordingly.
3
If someone was absent due to an approved leave, their leave status will reflect in the monthly view once leave is approved.

Manual Corrections

If a punch is wrong (wrong time, missed punch) and the employee cannot use the site portal, you can correct it directly in Supabase → attendance table. Edit the check_in, check_out, or status field for that emp_id + date record. Always note the reason in your HR log.

Step 6 — Monthly Salary Processing

Run this process between the 1st and 5th of each month for the previous month.

Attendance & Salary › Salary Sheet tab
1
Select the correct Month and Year. Click Generate Salary Sheet.
2
ERP calculates for each employee: Working Days, Present Days, Half Days, Absent (unpaid), Gross, OT Pay, PF, ESIC, PT, TDS, Advance Deductions, Net Wages.
3
Review the TDS column — adjust TDS amounts if your CA has given updated monthly figures. (Click directly in the TDS cell to edit.)
4
Log any salary advances in the Advances tab before generating (they auto-deduct).
5
Click Post to Financials to record the salary in the Finance system.
6
Click Export CSV to download the salary sheet — share with accounts for bank transfer processing.
7
Click Generate Salary Slips → Print/Save each slip as PDF → distribute to employees (WhatsApp is fine).

Checklist Before Finalising Salary

  • All attendance for the month is correct — no missing punches or wrong statuses
  • Leaves are approved/rejected in Leave Tracker
  • Advances for the month are entered in the Advances tab
  • Any new joiners or exits during the month have their dates correctly set
  • TDS amounts confirmed with CA (if applicable)
  • OT hours look reasonable — spot-check 2–3 workers
Never post to financials until you've verified the attendance data. Once salary records are saved, corrections require manual Supabase edits. A few minutes of verification saves a lot of trouble.

After Salary is Run

  • Keep a printed copy of the salary sheet in the monthly compliance folder (one folder per month).
  • File individual salary slips in each employee's physical file (or note that slip was sent digitally).
  • Prepare PF and ESIC payment challans using ERP figures — remit by 15th.
  • Prepare PT payment — remit before 7th.

Step 7 — Increments & Letters

Giving an Increment

1
In HR Dashboard → Employees, open the employee's card. Update the Monthly Salary field to the new CTC. Save.
2
Sync to Attendance: go to Attendance → Employees → ↻ Sync from HR. The new salary is now active from the next salary run.
3
Click 📈 Increment Letter in the employee's HR card. Enter the previous CTC when prompted. The letter is generated with old CTC, new CTC, increment amount, and percentage.
4
Print the letter → get employee signature on office copy → file it → send employee their copy.
Best Practice: Update the salary in ERP after the last salary run of the old CTC month. This way the old month is processed at the old rate and the new month starts at the new rate automatically.

Step 8 — Exit Process

When an Employee Leaves

1
Collect resignation letter in writing. File it immediately.
2
Note the last working day. Ensure notice period is served or notice pay recovery is agreed.
3
Run a No-Objection checklist: PPE, tools, ID card returned; no outstanding advances; all dues settled.
4
Process the Full & Final salary in the Salary Sheet for the exit month (attendance up to last working day is auto-tracked).
5
In HR Dashboard → employee card, update Status to Resigned or Terminated. This removes them from future salary runs.
6
Run ↻ Sync from HR in Attendance — they will be marked inactive and excluded from future sheets.
7
Delete from ZKTeco machine: User Management → find user by ID → Delete. This prevents ghost punches.
8
Issue Relieving Letter and Experience Letter within 15 days. File office copies.
9
Retain the employee's physical file for a minimum of 5 years after exit (labour law requirement).
